# Finaryo Frontend

React + TypeScript UI for Finaryo.

## Commands

- `npm run dev` - start Vite dev server
- `npm run build` - type-check + production build
- `npm run preview` - preview built app
- `npm run lint` - lint frontend files

## Notes

- API base URL is controlled by `VITE_API_BASE_URL` (optional).
- If not set, frontend uses relative API paths (works with monolith backend on the same host/port).
